## 🔧 发现的问题

### 1. Token Schema 缺少 user 字段
- **问题**: 登录接口未返回用户信息
- **修复**: 在 `app/schemas/user.py` 中添加 `user` 字段到 `Token` 类
- **状态**: ✅ 已修复

### 2. 缺少 expires_in 字段
- **问题**: 登录响应未包含 token 过期时间
- **修复建议**: 在 `app/services/user_service.py` 的 `login` 方法中添加
- **状态**: ⚠️ 待修复（不影响功能）
