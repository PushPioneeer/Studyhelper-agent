# 错题集和追问功能实现总结

2026-03-18

## 🎯 实现目标
1. **错题自动保存**: 所有拍照解题的任务自动保存到错题集

## ✅ 功能实现

### 1. 数据模型层

#### 错题模型 (`app/models/mistake.py`)
```python
class Mistake(Base):
    """错题模型"""
    - 题目内容（文本、图片、科目、题型）
    - 答案信息（正确答案、用户答案、解析）
    - 知识点（JSON 格式）
    - 错误分析（错误类型、错误分析、难度）
    - 状态（new/reviewing/mastered/archived）
    - 复习记录（复习次数、最后复习时间）
```

#### 追问记录模型 (`app/models/follow_up.py`)
```python
class FollowUp(Base):
    """追问记录模型"""
    - 对话内容（用户提问、AI 回复）
    - 上下文（父消息 ID、对话顺序）
    - 元数据（创建时间、IP 地址）
```

### 3. 服务层

#### 简单解题服务 (`app/services/simple_question_solver.py`)

**新增功能**:
1. **错题自动保存**
   ```python
   async def _save_wrong_question(
   ```


3. **追问功能（基于错题记录）**
   ```python
   async def answer_follow_up(
       self,
       user_id: int,
       question_id: int,
       follow_up_question: str
   ) -> str:
       """回答追问（基于错题记录）"""
   
   async def answer_follow_up_stream(
       self,
       user_id: int,
       question_id: int,
       follow_up_question: str
   ) -> AsyncGenerator[str, None]:
       """回答追问 - 流式输出版"""
   ```

**实现细节**:
- 使用内存数据库 (`Dict[int, List[Dict]]`) 存储错题
- 支持按科目筛选错题
- 支持流式输出追问回答
- 自动从错题记录中获取题目信息用于追问上下文

## 📝 使用示例

### 1. 上传题目（自动保存错题）
```python
import httpx

async with httpx.AsyncClient() as client:
    with open("question.png", "rb") as f:
        files = {"images": ("question.png", f, "image/png")}
        data = {"subject": "math"}
        
        response = await client.post(
            "http://localhost:8000/api/v1/questions/upload",
            files=files,
            data=data,
            headers={"Authorization": f"Bearer {token}"}
        )
        
        question_id = response.json()["id"]
```

### 2. 获取错题集
```python
response = await client.get(
    "http://localhost:8000/api/v1/questions/wrong-questions/",
    headers={"Authorization": f"Bearer {token}"}
)

wrong_questions = response.json()["data"]
```

### 3. 获取错题详情
```python
response = await client.get(
    f"http://localhost:8000/api/v1/questions/wrong-questions/{question_id}",
    headers={"Authorization": f"Bearer {token}"}
)

detail = response.json()["data"]
```

### 4. 追问功能
```python
response = await client.post(
    "http://localhost:8000/api/v1/questions/follow-up/new",
    json={
        "question_id": question_id,
        "question": "为什么选项 C 不是？"
    },
    headers={"Authorization": f"Bearer {token}"}
)

answer = response.json()["data"]["answer"]
remaining = response.json()["data"]["remaining"]
```

### 5. 流式追问
```python
async with client.stream(
    "POST",
    "http://localhost:8000/api/v1/questions/follow-up/new/stream",
    json={
        "question_id": question_id,
        "question": "为什么选项 C 不是？"
    },
    headers={"Authorization": f"Bearer {token}"}
) as response:
    async for line in response.aiter_lines():
        if line.startswith("data: "):
            data = json.loads(line[6:])
            if "chunk" in data:
                print(data["chunk"], end="", flush=True)
            elif "done" in data:
                print(f"\n追问次数：{data['follow_up_count']}")
```
